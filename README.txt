# RaymondMarioCharles.github.io

Welcome to my personal website repository! This is the source code for my **personal portfolio** built using **HTML**, **CSS**, and **JavaScript**. It also includes some **interactive elements** such as a chat feature, integrated with **Spring Boot** and **WebSockets**.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Spring Boot, WebSocket
- **Deployment**: GitHub Pages (for the website)

## Project Setup

### Clone the Repository
To get started with this project locally, you can clone this repository using the following command:

git clone https://github.com/RaymondCharles/RaymondMarioCharles.github.io.git
